# 关于我们

[← 返回主页](../README.md)

FarmMaster 农场大师是一个完成本地化的 Web3 项目交互程序，支持多个 Web3 项目的协议交互脚本及浏览器自动化脚本，同时还为会员提供一系列链上工具，实现用户多账号一键批量完成空投交互任务。

## 产品特色

- 已支持 25 + 热门 Web3 项目的自动化交互
- 通用的银河任务模块，一键获取、完成银河任务及注册绑定
- 浏览器自动化脚本，实现批量任务自动化交互
- 一系列链上工具，提高交互效率
- 技术客服支持
