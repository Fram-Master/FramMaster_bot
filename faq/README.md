## 脚本运行

**问：运行时报错429是怎么回事？**

答：出现这种问题建议再重新跑一次，如果多次不行大概率是代理IP不行。

## 程序安装和更新

**问：我点更新后程序提示下载完成并重启了，但还是反复提示更新怎么办？**

答：更新失败了，一般是网络问题，尝试重新下载。

**问：我是windows/Mac用户，更新导致黑屏怎么解决？**

答：群内 @孤风 说明一下系统，电脑版本。
