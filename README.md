## 💡 核心功能

### 项目交互
- **Irys项目**：自动领水、玩游戏、上传文件、Mint NFT
- **银河任务**：自动完成银河平台任务并领取奖励
- **多链支持**：支持EVM、Solana等多个区块链

### 自动化工具
- **指纹浏览器**：支持多账号管理
- **代理IP管理**：动态代理支持
- **打码服务**：在线和本地打码支持
